Bot Maker

This is a simple guide to deploy our project Campus Bot to any campus.



Firstly you have to create a QnA Maker application at qnamaker.ai. Create an application and feed it the URL's of your FAQ pages so that the bot can smartly answer user queries. When this is done please paste the knowledge base id and Subscription key below.

Now we will configure LUIS (Language Understanding Intelligent Service) which allows us to understand natural language instructions easily. For this we provide a pre-trained model to you. Just click the button to download it. Next go to luis.ai and create a new application. Click import model and upload the downloaded file. After this just copy and paste the LUIS App Id and Endpoint Key to this form.
One feature of the bot is to alert it's users of events happening around them. To do this the bot crawls a given set of Facebook Pages to get the relevant events. Please enter the Page Ids of the pages that you would like your bot to crawl (comma separated).
Finally you have to upload the Student and Course Databases, these databses are to be submitted as csv files and we will convert them into JSON data for use.
